# Github Action Examples

Sample examples of Github Action

The yaml structure is ideally in below format.

````yaml

Workflow:
  * on Event:
    * Job-1:
      * Step-1:
      * Step-2:
    * Job-2:
      * Step-1:
      * Step-2:
````
